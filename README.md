### Dependencies

- fish > 3.1.0
- A nerd font. `Iosevka NF` is recommended.

#### Install

```fish
fisher install projekt0n/biscuit
```
To install fisher. follow the installation guide for fisher ![here](https://github.com/jorgebucaran/fisher) in the fisher repo.
