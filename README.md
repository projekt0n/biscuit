# Biscuit

A minimal fish shell prompt.

![Screenshot](https://i.imgur.com/n9YvfXy.png)

### Dependencies

- fish > 3.1.0
- A nerd font. `Iosevka NF` is recommended.

#### Install

```fish
fisher install projekt0n/biscuit
```
